# Sales Dashboard — Power BI

## Overview
Interactive Sales Dashboard built in Power BI analyzing retail sales data 
across products, categories, payment modes, and sale types.

## Key Metrics
- Total Sales: 364.15K | Profit: 32K | Profit%: 9.52%

## Features
- Sales & Profit trend by Month
- Payment Mode split (Online vs Cash)
- Sales by Product, Category & Sale Type
- Daily Sales volatility analysis
- Year filter (2024 / 2025) + Month slicer

## Tools Used
Power BI Desktop | DAX | Data Modeling
